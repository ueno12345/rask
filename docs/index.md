# 使用方法

## 新規ユーザ登録
画像右上の"新規登録"をクリックすると，新規ユーザ登録ページへ移動する．

<img src="images/resistration.PNG" width="50%">

## タスク・ユーザ管理
画像左上の"タスク一覧"または"ユーザ一覧"をクリックすると，タスク一覧ページまたはユーザ一覧ページへ移動する．

<img src="images/task.PNG" width="40%"> <img src="images/user.PNG" width="40%">

## 新規タスク作成
タスク一覧ページで"新規作成"をクリックすると，新規タスク作成ページへ移動する．
